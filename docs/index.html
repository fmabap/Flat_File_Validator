JSON File Structure Schema The JSON File Structure Helper helps you to generate the JSON File Structure.
You have to enter the record type field position (starting from 1) and the length.
Additional you have to enter the record types descriptions in the text area. Each attribute is separated by tab each field of the record needs to be entered on a separated line
You have to press ALT + 09 on your numpad of the keyboard to add a tab in the textarea. So it is easier to copy the values from an excel sheet like this one 1. Record Type Id (name) 2. Field Id (name) 3. Field Length 4. Is the field obligatory (true or X / false), optional, default = false 5. Allowed values of the field in array style, optional, example: ["Value A", "Value B"] 6. A regex that the field must match in JSON style escaped, optional, example: ^T.*$
